#### **Image-Filters**
Quick image filters

A console-based image filter. 

Supported filters include:
 - BLUR
 - CONTOUR
 - DETAIL
 - EDGE_ENHANCE
 - EDGE_ENHANCE_MORE
 - EMBOSS
 - FIND_EDGES
 - SMOOTH
 - SMOOTH_MORE
 - SHARPEN

You can upload image from <code>**url**</code> or from you computer using an absolute <code>**path**</code>.
